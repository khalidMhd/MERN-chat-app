### MERN CHAT APPLICATION
# Full functional real time chat app with MERN Stack.. 


** Install dependency **
Download project clone and open with cmd

** cleint **
cd client - npm install & npm run

** Server **
cd api - npm install & npm run

** socket **
cd cocket - npm install & npm run
